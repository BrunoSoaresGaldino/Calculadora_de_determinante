
Este programa, calcula o determinante de uma matriz nxn.O método usado para o cálculo do determinante é o 
Teorema de Laplace.

A entrada é dado por um arquivo de texto puro, vide exemplo no diretório examples. A saída pode ser tanto na
tela quanto em um artuivo.

Uso

	Na linha de comando digite: determinante <arquivo_entrada>, para saída na tela ou determinante <arquivo_entrada>
 <arquivo_saida>, para saída em arquivo.
